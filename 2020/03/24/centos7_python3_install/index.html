在CentOS7上安装Python3

发表于 2020-03-24 分类于 学习笔记
本文字数： 1.2k 阅读时长 ≈ 2 分钟

  1. 安装环境基础依赖包

    linux注重系统的安全和稳定，所以必需先安装环境的依赖包才能使用环境。

  2. 新建一个用户，进入用户家目录下，创建Python3文件夹

    防止在对Python环境进行操作时，影响系统环境，故需要新建一个用户，在普通用户下安装Python。

  3. 利用SecureCRT等工具将Python3包(Gzipped source tarball)上传至CentOS并解压

    SecureCRT下载地址，Python3下载地址

  4. 进入解压后的目录，进行编译

    注意需要带上ssl模块

  5. 配置环境变量

    需要切换回家目录，并在bashrc的末尾增加配置

    按方向键↓至末尾后，按O，插入以下配置

    输入完成后按ESC，输入:wq保存并退出

  6. 重新加载bashrc文件

  7. 配置完成

    注意运行的命令是python3而不是python（因为linux系统自带python2，使用python默认运行的是python2）
